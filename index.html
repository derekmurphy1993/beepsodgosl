Tic Tac Toe

General Assembly

Tic Tac Toe is a single page javascript application for a user to play against themselves. Utilizing an API the user is able to authenticate a sign in, play a game that will allow them to place moves, and calculate which player won or if it is a draw. The user can also see how many games they have played.
